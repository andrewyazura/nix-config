## Instructions

### yorha2b - main pc

1. Copy ssh keys to `~/.ssh`
2. Apply correct permissions:

```bash
chmod -R 700 ~/.ssh/
chmod 600 ~/.ssh/*
chmod 644 ~/.ssh/*.pub
```

3. Clone nix-config repo

```bash
nix-shell -p git
git clone git@github.com:andrewyazura/nix-config.git
```
